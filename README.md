# SOP-AI-Training-System
An AI-inspired SOP training system that converts raw SOP documents into structured summaries, step-by-step learning content, and assessment questions. Designed as a practical workflow automation solution using Google Apps Script.
# SOP → AI Training System

## Overview
This project converts SOP (Standard Operating Procedure) into structured training content.
Full working Link:- https://script.google.com/macros/s/AKfycbxZyhjohnd3CsY_ULtSuwjOCNX1tKBIWie3LeGVrtikjfWiurZh_1uu7FS8Gopk3khdlA/exec

## Features
- Structured Summary
- Step-by-step Training Guide
- Quiz Questions
- File Upload (.txt)
- Clean UI

## How It Works
- User inputs SOP
- System processes text
- Generates structured output

## Tech Stack
- Google Apps Script
- HTML, CSS, JavaScript

## Note
This project demonstrates AI-like processing logic. In production, this can be integrated with real AI APIs (OpenAI / Hugging Face).
